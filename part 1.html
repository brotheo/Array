IGCSE Computer Science

Algorithm Design Question

A program is required to manage the results of a competition between a current champion and a challenger.

The program must:

  • Input the name and score of the challenger.
  • Input the name and score of the current champion.
  • Input the highest score ever recorded.
  • Calculate the average score of the challenger and the champion.
  • Display the average score.

The program must then determine:

  • If the challenger becomes the new champion.
  • If the highest score record has been beaten.

The program must output appropriate messages based on these conditions.

Task

Write an algorithm using pseudocode that:

  1. Declares all necessary variables.
  2. Inputs all required data.
  3. Calculates and outputs the average score.
  4. Uses selection statements to determine and display the correct results.
